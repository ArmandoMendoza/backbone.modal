backbone.modal.js

A plugin for Backbone that removes boilerplate code and simplifies creating modals for your application.

In many apps, modals are an essential part of the interaction. Especially with Backbone, you can create specific modals to handle data or provide information without leaving the 'page'.

Download v0.6 Source on Github

Getting Started

If you'd like to try out Backbone.Modal, be sure to read the docs. Check out the examples to know what kind of modals you can create. Once you're ready, just add the plugin code to your project and you're all set.

New to Backbone? Please check out Backbone's site and make sure you understand Backbone.Views.
